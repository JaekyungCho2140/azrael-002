## 1. Phase 2 목적

Phase 0에서 계산된 일정 테이블을 기반으로 **이메일 본문 텍스트를 자동 생성**합니다.

**주요 가치**:
- 협력업체/유관부서에 일정 공유 이메일 작성 시간 80% 절감
- 일관된 형식의 전문적인 이메일 생성
- 복사-붙여넣기로 즉시 발신 가능

---

## 2. 기능 개요

### 2.1. 이메일 텍스트 생성

**입력**: 테이블 1 (일정표)
**출력**: 이메일 본문 텍스트 (HTML 또는 Markdown)

**예시**:
```
제목: [L10n] 26-02-10 업데이트 일정 안내

안녕하세요,

다음과 같이 2026년 2월 10일 업데이트 현지화 일정을 안내드립니다.

□ 헤즈업: 01/28(화)
□ iOS 심사일: 02/03(월)

[일정표]
┌────┬────────┬──────────────┬──────────────┬────────┐
│ #  │  배치  │    마감      │  테이블 전달 │  설명  │
├────┼────────┼──────────────┼──────────────┼────────┤
│ 1  │  정기  │ 01/10(금) 09:00 │ 01/15(수) 18:00 │        │
│ 2  │  1차   │ 01/20(월) 09:00 │ 01/25(토) 18:00 │        │
│ 3  │  2차   │ 02/01(일) 09:00 │ 02/05(목) 18:00 │        │
└────┴────────┴──────────────┴──────────────┴────────┘

※ Disclaimer: {프로젝트 Disclaimer 내용}

문의사항은 회신 주시기 바랍니다.

감사합니다.
L10n팀 드림
```

### 2.2. 이메일 템플릿

**형식**: 사용자 커스터마이징 가능한 템플릿
**변수**: `{업데이트일}`, `{헤즈업}`, `{iOS 심사일}`, `{테이블}`, `{Disclaimer}` 등

**템플릿 예시**:
```
제목: [L10n] {업데이트일} 업데이트 일정 안내

안녕하세요,

{업데이트일} 업데이트 현지화 일정을 안내드립니다.

□ 헤즈업: {헤즈업}
□ iOS 심사일: {iOS 심사일}

{테이블}

※ {Disclaimer}

감사합니다.
L10n팀 드림
```

### 2.3. UI 플로우

```
[테이블 1]
  ↓
[이메일 생성] 버튼
  ↓
[이메일 미리보기 모달]
  ├─ 템플릿 선택 (기본/간단/상세)
  ├─ 본문 편집 (선택적)
  └─ [복사] 또는 [Gmail로 전송]
  ↓
[복사] → 클립보드 복사 → "복사되었습니다" 메시지
또는
[Gmail로 전송] → Gmail 작성 화면 자동 열기 (mailto 또는 Gmail API)
```

## 3. 기술 스택 (예상)

| 기술 | 용도 | 비고 |
|------|------|------|
| 템플릿 엔진 | 변수 치환 | Handlebars 또는 직접 구현 |
| Clipboard API | 텍스트 복사 | `navigator.clipboard.writeText()` |
| Gmail API (선택) | Gmail 작성 화면 자동 열기 | 또는 `mailto:` 링크 |

## 5. 예상 사용자 스토리

**스토리 1**: 이메일 본문 자동 생성
- **As a** L10n 팀원
- **I want to** 계산된 일정을 이메일 본문으로 자동 생성하고 싶다
- **So that** 협력업체에 일정을 빠르게 공유할 수 있다

**스토리 2**: 이메일 템플릿 커스터마이징
- **As a** L10n 팀장
- **I want to** 이메일 템플릿을 프로젝트별로 커스터마이징하고 싶다
- **So that** 프로젝트 특성에 맞는 문구를 사용할 수 있다
