## 사용 가이드

### 템플릿 선택 방법

1. **프로젝트 규모 평가** (analyze-scope.md 참조)
   - 소규모 → 단일 PRD 템플릿
   - 중규모 → Master + Feature 템플릿 (Shared 선택적)
   - 대규모 → Master + Shared + Feature 템플릿 (전체)

2. **템플릿 복사**
   - 해당 템플릿 전체를 복사
   - 프로젝트명과 기본 정보 입력
   - `[...]` 표시된 부분을 실제 내용으로 대체

3. **불필요한 섹션 제거**
   - 프로젝트에 해당하지 않는 섹션은 제거
   - 예: UI가 없으면 "UI/UX" 섹션 제거

4. **필요한 섹션 추가**
   - 템플릿에 없지만 필요한 섹션 추가
   - 일관된 형식 유지
